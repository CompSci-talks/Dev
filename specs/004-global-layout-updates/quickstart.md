# Quickstart: Global Layout Updates

## Overview
This feature introduces a global `FooterComponent` and a `ScheduleComponent` layout block for the CompSci Talks platform.

## Development Setup

1. **Prerequisites**: Ensure Angular CLI is installed and dependencies are installed (`npm install`).
2. **Local Run**: Execute `npm start` to run the development server.
3. **Verification**: 
   - Navigate to `http://localhost:4200` to see the new Footer.
   - Click the "Schedule" link in the navigation bar to see the new Schedule page layout.
